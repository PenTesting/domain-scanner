# domain-scanner [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url] [![Coverage percentage][coveralls-image]][coveralls-url]
> A node utility to scan a domain with various techniques.

## Installation

```sh
$ npm install --save domain-scanner
```

## Usage

```js
const domainScanner = require('domain-scanner');

domainScanner('Rainbow');
```
